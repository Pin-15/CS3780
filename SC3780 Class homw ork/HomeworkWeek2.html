  • <html>: Root of an HTML document.
  • <head>: Contains meta-information (title, links).
  • <title>: Sets the document's title (shown in the browser tab).
  • <body>: Holds the main content (text, images, links).
  • <h1> - <h6>: Headings, with <h1> as the highest level.
  • <p>: Defines a paragraph.
  • <a>: Creates a hyperlink.
  • <img>: Embeds an image (src specifies the source).
  • <ul> and <li>: Creates unordered lists; <li> is for list items.
  • <div>: A generic container for grouping elements.
